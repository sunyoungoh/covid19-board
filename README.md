# Covid19-Board
OpenAPI를 활용해 Covid19 데이터를 받아와 전세계, 국가별 현황을 보여주는 `Vue3 + TS` 프로젝트입니다.
`반응형 웹사이트`로 PC, 태블릿, 모바일에서 이용할 수 있습니다.
<br>

<br>

<br>

## ✨ Live Demo (Responsive Web)
https://sunyoungoh.github.io/covid19-board/
<br>
`오픈 API 중단으로 더 이상 데이터를 받아올 수 없습니다 😢`

<br>

<br>

## 🪄 Tech Stack
* Vue3

* Vuex
